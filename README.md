# Jazz Book

One-page, mobile-first web app for iPhone that embeds a visible YouTube player and provides audiobook-style controls above it.

## Files

- `index.html`
- `styles.css`
- `app.js`

## Version 1

- Load a YouTube URL or video ID.
- Play or pause with custom controls.
- Seek backward or forward by 10 or 30 seconds.
- Show the current playback time.
- Add, jump to, and delete bookmarks.
- Persist the last video, bookmarks, and last playback position in `localStorage`.
- Copy, share, and paste app state between devices without deleting existing local data.
- Open `/shorts/` links in a taller vertical player layout.
- Treat videos shorter than 1:50 as vertical after the player reports duration.
